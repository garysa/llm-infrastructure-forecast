# LLM Forecast Project

## Purpose
Discussion notes and analysis on LLM infrastructure trends.

## Topics Covered
1. TPU (Tensor Processing Unit) - Google's custom ML accelerator
2. ASIC vs GPU dominance in AI compute
3. Enterprise LLM deployment patterns (hybrid private/public model)

## Files
- `llm_forecast_discussion.md` - Full discussion in markdown
- `llm_forecast_discussion.pdf` - PDF version
- `generate_pdf.py` - Script to regenerate PDF (uses reportlab)

## Regenerate PDF
```bash
python3 generate_pdf.py
```
